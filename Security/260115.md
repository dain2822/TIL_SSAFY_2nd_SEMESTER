## 개인정보유출 사고 사례

### 2011년 SK 컴즈 개인정보 유출 사례

: 2011년에 발생한 국내 최대 규모(3500만 건) 개인정보 유출 사건

: 직원들이 안전이 확인되지 않은 개인용 프로그램을 무분별하게 사용한 것이 원인(내부 PC 62대 감염)

: DBA PC가 해킹되어 사실상 DB Dump 수준 유출

### 2025년 SKT 해킹 9.7GB 분량 유출

: 장비간 비정상적으로 이동한 데이터 분량이 9.7GB로 확인

: USIM 관련 텍스트 데이터라면 사실상 전체 데이터가 모두 유출된 것으로 추정

- 4G EPC HSS가 가지고 있는 정보
    - HSS(Home Subscriber Server)는 가입자 핵심 정보를 담고 있는 중앙 데이터베이스
    - 유출 가능한 정보
        - IMSI(가입자 식별 고유번호) 및 인증 키(세션 키 생성)
        - 가입자 프로필(데이터 요금제 속도, SMS 가입 여부, 로밍)
        - 최근 위치 정보(UE의 최근 접속 MME)
    - 개인 식별 정보(이름, 주민등록번호나 주소 등)는 저장하지 않는 것이 일반적임
- BPFdoor의 주요 특징
    - 조작된 패킷을 사용해 방화벽 우회 가능
    - Berkeley Packet Filtering 기술을 사용해 소켓 통신이 아닌 로우 패킷 직접 조작 방식으로 통신
        - 커널에서 작동하는 Rootkit 스타일 악성 코드
    - Reverse shell을 제공해 시스템 원격제어 가능
        - 하드코드된 패스워드를 이용해 공격자 인증
    - 패킷을 직접 조작하는 방식이므로 일반 소켓통신이 아님
        - TCP, UDP, ICMP 모두 가능
        - 포트 스캔으로는 탐지 불가
