- Spring, Spring Boot
    - Spring
        
        : 자바 기반의 대표적인 애플리케이션 프레임워크로, 객체 생성과 의존성 관리를 지원하는 DI와 IoC 컨테이너를 핵심으로 함
        
        : 개발자가 필요한 라이브러리를 직접 선택하고 설정 파일을 통해 환경을 구성해야하기 때문에 비교적 설정 과정이 복잡하지만, 그만큼 세밀한 제어가 가능함
        
    - Spring Boot
        
        : Spring 프레임워크를 보다 쉽고 빠르게 사용할 수 있도록 만든 확장 프로젝트
        
        : 자동 설정 기능이 가장 큰 특징
        
        : 의존성을 추가하면 기본 설정을 자동으로 구성해주어 개발자가 직접 XML이나 설정 클래스를 많이 작성하지 않아도 됨
        
        : 내장 톰캣과 같은 WAS를 포함하고 있어 별도의 서버 설치 없이 실행 가능한 애플리케이션을 만들 수 있음
        
    
    : Spring은 설정을 직접 구성하는 대신 높은 자유도를 제공
    
    : Spring Boot는 자동 설정과 내장 서버를 통해 개발 생산성을 높인 Spring 기반의 편의성 중심 프레임워크
    
    : 실무에서는 초기 설정 시간을 줄이고 빠르게 서비스를 개발할 수 있기 때문에 현재는 대부분 Spring Boot를 사용하는 추세
