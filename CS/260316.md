- CDN(Content Delivery Network)
    
    : 사용자에게 콘텐츠를 빠르게 제공하기 위해 여러 지역에 분산된 서버를 통해 콘텐츠를 전달하는 네트워크 시스템
    
    : 사용자와 가까운 서버에서 데이터를 제공하여 지연 시간(Latency)을 줄이고 서버 부하를 분산하는 기술
    
    : 이미지, 영상, 정적 파일(JS, CSS) 등을 빠르게 전달하기 위해 많이 사용됨
    
    - 필요한 이유
        
        : 사용자 → 가장 가까운 CDN 서버(Edge Server) → 콘텐츠 제공
        
        ⇒ 응답 속도 향상
        
    - 동작 방식
        - 원본 서버(Original Server)에 콘텐츠 저장
        - CDN 서버가 콘텐츠를 캐싱
        - 사용자는 가장 가까운 CDN 서버에서 콘텐츠 다운로드
    - 주요 구성 요소
        - Origin Server
            
            : 실제 콘텐츠가 저장된 원본 서버
            
        - Edge Server
            
            : 사용자와 가까운 위치에서 콘텐츠 제공하는 CDN 서버
            
        - Cache Server
            
            : 자주 요청되는 데이터를 저장
            
    - 장점
        - 웹사이트 응답 속도 향상
        - 서버 부하 감소
        - 트래픽 분산
        - 대규모 트래픽 처리 가능
        - DDoS 공격 완화 효과
    - 단점
        - CDN 사용 비용 발생
        - 캐시 갱신 관리 필요
        - 동적 콘텐츠에는 제한적
